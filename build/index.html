Beagle.prototype.v1

This is a mock of what the interface looks like with the current React templates. Please comment by posting an issue.

Milestone 1: Plugin Open

The plugin is opened (clicking on the extension icon).

Milestone 2: Hidden - Margins

The plugin starts out hidden, displaying only small margin annotation indicators.
Margin annotations will be added in Milestone 7.

Milestone 3: Author Profile

Upon clicking any person's name, an Author Profile pops up.

Milestone 4: Publication Graph

Upon clicking any person's name, an Author Profile pops up.

Milestone 5: Tags

Tags are associations made by either you, your field, or everyone. The key will be in being able to filter what tags are viewed. Here, I've denoted the scope of tags with different background colors.

Milestone 6: BibTeX

Give convenient access to BibTeX + other bibliography systems. Do whatever others already do.

Milestone 7: Saved Papers

Access your saved papers, and papers from your colleages.

Milestone 8: Annotations

While viewing annotations, highlight the text and display hypothesis style notes (notes NOT shown here).

Milestone 9: Journal Information

Access information about Journals and the current Issues

Milestone 10: User Authentication

On trying to save, annotate, or take notes, the user is prompted to either sign in to a persistent session, or to sign up using their email and a password (or, perhaps, we simply email them the password, and they set a new one each time by checking their email, and we don't store anything). This temporarily removes the sidebar.

Milestone 11: Paper Information

For papers that are not currently in the browser, but are linked to by Beagle.

Milestone 12: Notes

Save a high-level note about a paper when you save the paper
